# Known Limitations

## Issue with Certain HTML Tags

While the `Typography` component supports the `as` prop for dynamic tag rendering, certain HTML tags like `<span>` may not behave as expected with some text-align or layout-related styles due to their default CSS behavior.

For instance:

```tsx
<Typography as="span" textalign="right">
  This text might not align correctly
</Typography>
```

You can resolve this issue using one of these approaches or wrapping the `Typography` component in a parent element like `div`:

```tsx
<Typography as="span" textalign="right" style={{ display: "block" }}>You can use inline style</Typography>
<Typography as="span" textweight="light" className="block text-right">You can use Tailwind classes</Typography>
<Typography as="span" size="xl6" className="block text-right" style={{ color: "red" }}>You can combine both inline styles and Tailwind classes</Typography>
```
